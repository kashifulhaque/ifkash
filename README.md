# [**Kashiful Haque**](https://ifkash.dev) 🧑🏽
My personal website 👋

### **Tech stack** 📚
- [SvelteKit](https://kit.svelte.dev)
- [Bun](https://bun.sh)
- [Rust](https://rust-lang.org)
- [Cloudflare](https://www.cloudflare.com)

### **To run locally** 🏃🏼‍♂️
```sh
git clone https://github.com/kashifulhaque/ifkash;
cd ifkash;
bun install;
bun run dev;
```
